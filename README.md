# Todo List GUIアプリ（Python + tkinter）

## 概要
Pythonの標準GUIライブラリ「tkinter」を使った、シンプルなTodoアプリ。
タスクの追加・完了・保存・読み込みなどの基本機能。

## 特徴
- 締め切りと優先度の入力
- 完了状態の管理
- JSON形式でのデータ保存
- GUIベースの直感的な操作

## 使用方法
1. このリポジトリをクローン
2. `todo.py` を実行（Python 3.6以降）
